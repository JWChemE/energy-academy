HVAC commissioning focuses on flow, pressure, temperature, and control response.

## Flow balancing

Measure actual flow at each zone terminal (VAV box, radiator, diffuser) and adjust control valves to match design.

Typical procedure:
1. Measure supply flow at main fan outlet
2. Measure return flow at main return
3. Measure zone flows (sum should equal main)
4. Adjust zone balancing valves so each zone gets design flow
5. Verify no excessive pressure drop at any valve

Without balancing, low-resistance zones starve others, causing comfort complaints and control difficulty.

## Pressure drop verification

Measure pressure at key points and compare to design:
- Supply fan discharge pressure
- Duct static pressure (should be ~25–40 Pa target)
- Chiller inlet/outlet pressure drop
- Boiler inlet/outlet pressure drop

Excessive drop suggests undersized pipes, fouling, or control valve sizing issues.

## Sensor calibration

Check temperature sensors against a reference thermometer:
- Handheld pyrometer (surface temp)
- Calibrated immersion thermometer (fluid temp)

If sensor reads 2°C high, control is off by 2°C. Recalibrate or replace.

## Setpoint verification

Walk through each zone and confirm thermostat setpoints match design. Common errors: setpoints never adjusted from default, or occupants cranked them up/down (19°C instead of 21°C).

Document all setpoints as-built.
