**Performance testing** measures whether equipment delivers designed capacity and efficiency.

## Chiller performance

Design: 100 kW cooling at 6°C outlet, 12°C inlet (6°C approach).

Test procedure:
1. Run chiller at full load (simulate peak cooling demand)
2. Measure supply temp, return temp, flow rate
3. Calculate actual cooling: flow × Cp × ΔT
4. Compare to design

Acceptable range: ±10% of design.

If chiller delivers 85 kW instead of 100 kW (15% low), it's fouled, refrigerant charge is low, or the compressor is weak.

## Boiler performance

Design: 500 kW heat output.

Test procedure:
1. Measure fuel input (gas flow, oil consumption)
2. Measure outlet temp, flow rate, return temp
3. Calculate output: flow × Cp × ΔT
4. Calculate efficiency: output / input

Acceptable: Within 2–5% of nameplate efficiency.

If efficiency is poor, boiler may be fouled (scale), tuned poorly (excess air), or degraded.

## Seasonal variation

Equipment performance varies with outdoor conditions. Test in multiple seasons:
- Heating: coldest day
- Cooling: hottest day
- Mild: spring/fall (part-load performance)

Part-load performance is often worse than full-load because equipment is oversized and cycles inefficiently.

## Documentation

Record all test results in a commissioning log. These become the baseline for future performance monitoring.
