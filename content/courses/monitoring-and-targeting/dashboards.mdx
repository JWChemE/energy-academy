A **dashboard** turns data into action. Show:
1. Current consumption vs target
2. 12-month trend (normalised)
3. Consumption by system (if submeters)
4. vs forecast (accounting for expected weather)
5. Alerts (exceptions detected)

## Who uses it?

Facilities (daily check), Finance (monthly reporting), Leadership (annual progress), Whole building (public display).

## Tools

- Simple: spreadsheet (free, manual)
- Mid: data logger + cloud (£2,000–5,000)
- Advanced: BMS + M&T module (£10,000+)

Start simple, upgrade as you scale.

## Power

A visible dashboard (especially public) changes behaviour. People become aware, spot problems, take ownership. Simply displaying consumption (no commentary) reduces energy by 5–15%.
