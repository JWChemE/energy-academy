Your **baseline** is the reference point. Usually a normalised average consumption over one full year (covers all seasons).

## Good baseline qualities

Representative (all seasons), normalised (adjusted for weather/occupancy), recent (reflects current use), documented (assumptions clear).

## Calculating

1. Gather 12 months of meter data
2. Get HDD for each month
3. Regress: kWh = a + b × HDD
4. Normalise each month to average HDD (say, 100 HDD/month)
5. Average the 12 months → baseline

Raw consumption varies 5,000–10,000 kWh/month seasonally. Normalised baseline: ~7,500 kWh/month.

## Baseline updates

If building use changes (more occupancy, new equipment, extended HVAC hours), update the baseline. Don't compare apples-to-oranges.
