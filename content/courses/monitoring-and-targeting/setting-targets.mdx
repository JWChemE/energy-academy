A target is a commitment: "We will reduce energy to X by [date]."

## SMART targets

Specific, Measurable, Achievable, Relevant, Time-bound.

## How to set

1. Start with baseline
2. Identify achievable improvements (audit + funding):
   - Quick wins (controls, insulation, balancing): 5–10%
   - Medium-term (equipment): 10–20%
   - Long-term (fabric): 20–30%
3. Phase over time:
   - Year 1: −10%
   - Year 3: −15%
   - Year 5: −25%

## Growth accounting

If occupancy is rising 5%, set absolute consumption flat (neutral) or +2%, while intensity target is −5% (kWh per person).

## Communicating

Post targets publicly, track monthly, celebrate hits, investigate misses. A public commitment (e.g., "50% by 2030") is powerful if backed by a detailed action plan.
